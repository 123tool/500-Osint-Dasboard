## ⚡ 500 OSINT Dasboard

**500 OSINT SPY** adalah *Command Center* intelijen berbasis Node.js yang mengintegrasikan lebih dari **500++ Tools OSINT** ke dalam satu dashboard agresif. Menggunakan desain **Neo-Brutalist High-Fidelity**, platform ini diciptakan untuk efisiensi investigasi digital, *threat hunting*, dan forensik data.

> **Status:** Production Ready  
> **Framework:** Node.js + Express  
> **UI Style:** Neo-Brutalist (High-Contrast / Bold Border)

---

## 🚀 Fitur Utama

- 🔗 **500++ Managed Links**: Database tools OSINT terlengkap yang mencakup 33 kategori.
- 📦 **Modular Database**: Penambahan tools cukup melalui `tools.json` tanpa menyentuh kode UI.
- ⚡ **Lightweight Engine**: Sangat ringan, bisa berjalan lancar di **Termux**, VPS spek rendah, maupun Desktop.
- 📱 **Responsive Interface**: Tampilan tetap "sadis" di layar HP maupun monitor ultrawide.
- 🛠️ **Fast Filtering**: Navigasi kategori instan untuk pencarian alat investigasi yang presisi.
